# MAINK-TECH
A static website with 1500+ UI/UX templates, a drag-and-drop editor, and an AI agent to create and host sites.

## Features
- Browse and edit templates.
- AI-assisted website creation.
- Host on GitHub Pages.

## Setup
- Clone: `git clone https://github.com/your-username/maink-tech-web.git`
- Open `index.html` in a browser.

## Deployment
- Push to repo, enable GitHub Pages in Settings.
